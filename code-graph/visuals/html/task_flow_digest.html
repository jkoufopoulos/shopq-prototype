V2 Pipeline Stages

7-stage digest generation pipeline

1
Extraction
Extract structured entities from emails: flights, events, deadlines, notifications.
2
Importance Classification
Classify each entity as critical, time_sensitive, or routine based on content signals.
3
Temporal Decay
Apply time-based decay to importance scores. Near-term events get boosted, distant events decay.
4
Section Assignment
Assign entities to sections: NOW (today), COMING_UP (this week), WORTH_KNOWING (FYI).
5
Narrative Generation
Generate natural language summaries for each section using LLM.
6
HTML Rendering
Render digest cards with Gmail links using hybrid_digest_renderer and Jinja2 templates.
7
Verification
NarrativeVerifier checks for hallucinations - numbers, dates, names must exist in source emails.
