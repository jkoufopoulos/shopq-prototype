Auto-Organize Flow

Background service worker automation

1
Alarm Fires
Chrome MV3 alarm triggers based on configured interval (shopq_auto_organize_settings.intervalMinutes).
2
Load Settings
Service worker loads auto-organize settings and records session start for telemetry.
3
Fetch Threads
Query Gmail API for unlabeled threads. Skip if inbox is empty.
4
Classify (Backend)
POST /api/organize with batch. Backend runs: Rules ‚Üí LLM ‚Üí Importance ‚Üí Temporal ‚Üí Labels.
5
Apply Labels
Apply Gmail labels and archive emails. Update sender cache for 24hr reuse.
6
Trigger Digest
Set shopq_digest_pending=true. When user opens Gmail, digest will be generated.
7
Send Digest
Call generateAndSendSummaryEmail() to create and send the daily digest via Gmail API.
8
Update Timestamp
Update shopq_last_auto_organize_at in Chrome storage. Clear digest pending flag.
