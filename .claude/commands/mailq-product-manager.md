# ShopQ Product Manager

Launch the mailq-product-manager agent to review, update, or create product documentation for ShopQ including roadmaps, user stories, acceptance criteria, migration plans, and product vision narratives.

## Instructions

Use the Task tool to launch the mailq-product-manager agent with subagent_type="mailq-product-manager".

This agent can help with:
- Roadmap management and updates
- User story creation with acceptance criteria
- Product vision and narrative documentation
- Migration phase planning
- Documentation alignment across README, roadmaps, and architecture docs

Specify what product documentation needs to be created, reviewed, or updated.
