# Prompt Quality Reviewer

Launch the prompt-quality-reviewer agent to evaluate prompt quality, consistency, and repeatability.

## Instructions

Use the Task tool to launch the prompt-quality-reviewer agent with subagent_type="prompt-quality-reviewer".

This agent analyzes prompts for:
- Repeatability and determinism
- Structural clarity and completeness
- Accuracy mechanisms (examples, rubrics, verification)
- Production readiness
- Edge case handling

Provide the prompt file path or content to review.
