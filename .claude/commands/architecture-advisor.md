# Architecture Advisor

Launch the architecture-advisor agent for architectural guidance, design decisions, system-level recommendations, technology comparisons, and refactoring strategy.

## Instructions

Use the Task tool to launch the architecture-advisor agent with subagent_type="architecture-advisor".

This agent can help with:
- Evaluating architectural approaches and trade-offs
- Technology selection and comparison
- Refactoring strategy and migration planning
- System design reviews
- Scalability and performance considerations

Provide clear context about the architectural question or design challenge you're facing.
